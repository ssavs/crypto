CRYPTOCURRENCY & NFTS

Crypto is digital currency in which transactions are verified and records maintained by a decentralized system using cryptography, rather than by a centralized authority. NFT means non-fungible tokens (NFTs), which are generally created using the same type of programming used for cryptocurrencies. In simple terms these cryptographic assets are based on blockchain technology.
